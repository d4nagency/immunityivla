# SEO Research Plan for Immunity IV LA

Goal: Outrank local competitors with comprehensive, user-focused SEO.

1. Competitive Audit: top LA IV therapy blogs, content depth, linking, local signals.
2. Keywords: core + long-tail + FAQs (LA, WeHo, Santa Monica, etc.).
3. Content Strategy: pillar pages, cluster posts, FAQ expansion, local schema.
4. On-Page: optimized titles, descriptions, headings, internal linking.
5. Technical: site speed, images, Core Web Vitals, structured data.
6. Off-Page: citations, guest posts, local partnerships.
7. Measurement: GA4 goals, UTM tracking, SEO dashboards.
8. GitHub workflow: SEO branch, PR review, previews.
